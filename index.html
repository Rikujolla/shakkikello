Fast chess features

Fast chess is an application with a chess clock, chess in two-player mode or against Stockfish chess engine. The basic views are Chess clock, Chess board and Settings. The default view is Chess clock. You can save your favourite settings including the start view setting. With cover actions either Chess clock or Chess board can be initiated with your favourite settings

To play chess in two player mode or against Stockfish is started from the settings page. Contributions for translations can be made at www.transifex.com. Project name is shakkikello.

Donations

The default timer is 5 minutes for White and Black. Timer values can be adjusted by one minute interval. After pressing the start button the timer works with swipes or by pressing the timers as buttons. Screen switch-off time value is recommended to be increased depending the intensity of the game.
